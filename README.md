# Free Shipping Label Generator by Baiko 📦

A free, browser-based 4×6 thermal shipping label generator built for ecommerce sellers, D2C brands, warehouses, and small businesses.

Generate professional courier-ready labels in seconds for Amazon, Flipkart, Blinkit, warehouse dispatch, B2B shipments, and self-delivery.

---

## ✨ Features

- Generate 4×6 thermal printer ready shipping labels
- Multi-step easy label creation flow
- Sender & receiver information
- Order / invoice details
- Courier details
- AWB barcode generation
- Self-delivery support
- Multiple box support
- Different dimensions & weights per box group
- Auto Box numbering
  - Box 1/10
  - Box 2/10
  - ...
- Automatic total weight calculation
- Download labels as PDF
- Print-ready format
- Mobile responsive UI

---

## 📦 Multi Box Support

Create labels for complex shipments.

Example:

Shipment:
- 5 boxes → 50×40×30 cm → 12 kg each
- 2 boxes → 30×20×15 cm → 5 kg each

The tool automatically generates:

```
Box 1/7
Box 2/7
...
Box 7/7
```

with correct dimensions and weight.
